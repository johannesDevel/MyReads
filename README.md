# MyReads Project

This project was created for Udacity's [Become a React Developer Nanodegree](https://www.udacity.com/course/react-nanodegree--nd019).

The MyReads project, is a bookshelf app that allows you to select and
categorize books you have read, are currently reading, or want to read. The project uses React
and a client library that allows the user to persist information.

## Getting Started

### Prerequisites
To run the application npm has to be installed, you can download it under [Node.js](https://nodejs.org).

### Installing
Follow these steps to start the application:

* install all project dependencies with `npm install`
* start the development server with `npm start`
